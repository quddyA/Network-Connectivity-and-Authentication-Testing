# Network Connectivity and Authentication Testing

## Project Overview
This lab verifies communication between domain systems by testing network connectivity and authentication.

---

## Environment
- Windows Domain Controller
- Windows Client
- Azure Virtual Network

---

## Tools Used
- Command Prompt
- Ping
- Nslookup

---

## Commands Used
ping 10.0.0.4

nslookup domain.local

---

## Tasks Performed
- Tested connectivity to domain controller
- Verified DNS resolution
- Confirmed domain login authentication

---

## Result
Network communication and authentication were successfully verified.
